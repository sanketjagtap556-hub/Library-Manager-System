# 📚 Library Management System

A simple **Library Management System** built with **Python** as a beginner project. This application helps manage books and library members using a user-friendly console interface while storing data in JSON files.

## ✨ Features

* 📖 Add new books
* 🗑️ Remove books
* 🔍 Search for books
* 👤 Add and manage library members
* 📋 View available books
* 💾 Store data using JSON files
* 🖥️ Console-based interactive menu

## 🛠️ Technologies Used

* Python
* JSON
* Object-Oriented Programming (OOP)
